# 👨‍🍳 Smart Sous Chef

## **Smart Sous Chef** is a cloud-integrated web application designed to reduce meal-planning stress and food waste. Users can input available ingredients, and the app retrieves real-time recipes using the **Spoonacular API**.


## 🛠️ Requirements & Installation

To run this project locally, ensure you have **Python** installed.

### 1. Clone the repository

`git clone [https://github.com/daniandp/Smart-Sous-Chef.git](https://github.com/daniandp/Smart-Sous-Chef.git)`

cd Smart-Sous-Chef

### 2. Install Dependencies

You must install the following libraries to connect the backend with the cloud services. Run this command in your terminal:

`pip install flask python-dotenv requests`

### 3. Environment Setup (.env)
- Create a file named .env in the root directory of the project.
- Add your Spoonacular API Key using the following format:

SPOONACULAR_API_KEY=your_secret_key

### 4. Run the Application

`python app.py`
